Tensor Expression Language¶

Initially, TVM represents models in an Intermediate Representation called Tensor Expression. Models must be lowered to Tensor Expression describes a computation without specifying how it’s accomplished.

The Compute Function¶

Tensor Expression operators split the definition of a model into two pieces: * Compute Function: describes the computation, element-wise, typically as a Python lambda

function.

  • Schedule: describes how to perform the computation, including loop structure, strides, unrolling and any acceleration performed by intrinsics using tensorization or vectorization.
